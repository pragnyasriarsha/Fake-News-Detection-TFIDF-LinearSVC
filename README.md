# Fake News Detection Using TF-IDF and Linear SVC

## Overview
This project focuses on fake news detection using Natural Language Processing (NLP) techniques and Machine Learning models.

The system classifies news articles as:
- Fake News
- Real News

## Features
- Text preprocessing
- Tokenization
- Stopword removal
- Lemmatization
- TF-IDF vectorization
- Linear SVC classification
- Model comparison
- Hyperparameter tuning using Weights & Biases (W&B)

## Models Used
- Naive Bayes
- Logistic Regression
- Linear SVC

## Best Model
Linear SVC achieved:
- Test Accuracy: 99.92%
- Macro F1 Score: 0.9992

## Technologies
- Python
- Scikit-learn
- NLTK
- SpaCy
- Weights & Biases
